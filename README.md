# WhatsApp Post Studio

## Setup

1. Clone the repo
2. cp .env.example .env
3. Add your GROQ_API_KEY (free at console.groq.com)
4. Optionally add ANTHROPIC_API_KEY (paid)
5. npm install
6. cd client && npm install && cd ..
7. npm run dev
8. Open http://localhost:5173

## AI Providers
- Groq (Free): Get key at https://console.groq.com
- Claude (Paid): Get key at https://console.anthropic.com
